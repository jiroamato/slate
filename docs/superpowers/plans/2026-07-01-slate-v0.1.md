# Slate v0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** Ship slate v0.1 — a Python true-fork of mulch (git-native agent memory) with mulch store parity plus enforcement hooks (`slate setup claude`), two-tier priming, and a write-time dedup gate.

**Architecture:** Library-first: `src/slate/*.py` modules hold all logic; `src/slate/commands/*.py` are thin argparse handlers over them. `store.py` is the single choke point for store file I/O (UTF-8, `\n`, POSIX paths). Hook hot path (`slate hook *`) never imports PyYAML until injection is actually needed and is fail-open.

**Tech Stack:** Python ≥3.11, uv-managed, hatchling build. Runtime dep: PyYAML only. Tests: pytest. CI: GitHub Actions 3-OS matrix.

**Parity source:** mulch 0.10.7 (github.com/jayminwest/mulch), studied 2026-07-01. All constants below were read from its source — do not re-derive.

## Global Constraints

- Python ≥3.11; runtime dependency: **PyYAML only** (dev deps unrestricted).
- Distribution name `slate-memory`; console script `slate = slate.cli:main`; extras `semantic` and `mcp` exist as empty stubs.
- Exit codes everywhere: `0` ok · `1` unexpected · `2` validation/usage · `3` dedup-blocked · `4` lock timeout · `5` no store.
- `--json` on every command. Success envelope `{"ok": true, "command": "<name>", ...}`; error envelope `{"ok": false, "error": {"code", "message", "hint", "retry"}}` (hint/retry may be null). JSON printed with `indent=2`, `ensure_ascii=False`.
- On-disk store format: UTF-8, `\n` newlines, POSIX separators — hardcoded in `store.py`, the only module that opens store files. JSONL lines serialized `json.dumps(record, separators=(",", ":"), ensure_ascii=False)` (matches JS `JSON.stringify`).
- Record IDs: `mx-` + sha256(`"<type>:<idKey value>"`).hexdigest()[:6].
- `recorded_at`/`archived_at` timestamps: ISO-8601 UTC with milliseconds and `Z` suffix (`2026-07-01T10:30:45.123Z`). Single source: `schema.now_iso()`, overridable via env `SLATE_NOW` (determinism for tests/byte-identity CI).
- Store discovery: walk up from cwd to the git root (inclusive); at each dir prefer `.slate/` over `.mulch/`. Exit 5 with `slate init` hint when absent.
- Config: `slate.config.yaml` preferred, `mulch.config.yaml` accepted, same schema. Invalid config → exit 2 (except inside `slate hook *`, where fail-open wins).
- All `slate hook *` subcommands: catch every exception, exit 0, no output, append traceback to `<tempdir>/slate/hook-errors.log`.
- Hook perf: no-op `slate hook pre-tool` < 150ms locally, CI gate 500ms.
- Semantic commits (`feat:`, `test:`, `docs:`, `chore:`, `ci:`); one commit per green task step-group.

## Mulch parity constants (from source — copy exactly)

- Six types; per-type `(required, optional, dedupKey/idKey, summary)`:
  - `convention`: req `content`; idKey `content`; summary = truncate(content, 60)
  - `pattern`: req `name`,`description`; opt `files`; idKey `name`; summary = name
  - `failure`: req `description`,`resolution`; idKey `description`; summary = truncate(description, 60)
  - `decision`: req `title`,`rationale`; opt `date`; idKey `title`; summary = title
  - `reference`: req `name`,`description`; opt `files`; idKey `name`; summary = name
  - `guide`: req `name`,`description`; idKey `name`; summary = name
- Common optional fields: `id`, `classification` (required: foundational|tactical|observational), `recorded_at` (required), `evidence{commit,date,issue,file,bead,seeds,gh,linear}`, `tags[]`, `relates_to[]`, `supersedes[]` (pattern `^([a-z0-9-]+:)?mx-[0-9a-f]{4,8}$`), `outcomes[{status: success|failure|partial, duration, test_results, agent, notes, recorded_at}]`, `dir_anchors[]`, `owner`, `status` (draft|active|deprecated; `archived` only in archive files), `archived_at`, `archive_reason`, `supersession_demoted_at`, `anchor_decay_demoted_at`.
- `truncate(text, max=100)`: if len ≤ max return text; else if first `[.!?]\s` occurs before max, cut after that punctuation; else `text[:max] + "..."`.
- BM25: k1=1.5, b=0.75; tokenizer: lowercase, `re.sub(r"[^\w\s-]", " ", s, flags=re.ASCII)`, split on whitespace; IDF `log((N - df + 0.5)/(df + 0.5) + 1)`; only score > 0 returned; sort desc.
- Search confirmation boost: `score * (1 + boost_factor * len(outcomes))`, default `search.boost_factor: 0.1`, `--no-boost` disables. Confirmation stars = count of outcomes with status == "success".
- Locks: `<file>.lock`, O_CREAT|O_EXCL, retry 50ms, timeout 5s, stale >30s (mtime) auto-unlink. Readers never lock.
- Default config (mulch DEFAULT_CONFIG): version "1", domains {}, governance {max_entries 100, warn_entries 150, hard_limit 200}, classification_defaults.shelf_life {tactical 14, observational 30}.
- Prime tier weights: star 100, foundational 50, tactical 20, observational 10. Token estimate: `ceil(len(text)/4)`; default budget 4000.
- Staleness: foundational never; tactical/observational stale when age-days > shelf_life; age = floor((now - recorded_at)/86400s).
- Archive: `<store>/archive/<domain>.jsonl`, first line banner `# ARCHIVED — not for active use. Run \`slate restore <id>\` to revive.` — reader skips `#` lines. Archived records get `status:"archived"`, `archived_at`, `archive_reason` (`"stale"` for prune).
- `.gitattributes` line written by init: `.slate/expertise/*.jsonl merge=union` (appended, never overwrites; skip if present).
- Domain name rule: `^[a-zA-Z0-9][a-zA-Z0-9_-]*$`.
- ID resolution: accept `mx-abc123`, `abc123`, or prefix; unique prefix match ok; ambiguous → error listing matches; miss → error with `slate query` hint.
- Dir anchors: `normalize_dir_anchor` (backslashes→`/`, strip trailing `/`, strip leading `./`, `.`/empty → ""), write-time rejection of absolute paths and `..` segments; `file_lives_under_dir(file, dir)` = normalized equality or `startswith(dir + "/")`; `infer_dir_anchors(files, threshold=3)` = parent dirs of ≥3 changed files, deepest, sorted.
- Evidence auto-fill: `evidence.commit` from `git rev-parse HEAD` when not given; context files = union of staged+unstaged `git diff --name-only`, sorted.

## Slate deltas (spec overrides mulch)

- Exit codes 0–5 (above) replace mulch's 0/1.
- JSON envelope `{ok, ...}` replaces mulch's `{success, command, ...}`.
- Tolerant reader: malformed JSONL line → skip + warning (never throw); unknown-type record → preserved/primed/searched, doctor notice.
- `record` dedup gate: BM25 self-query over target domain; top score ≥ `dedup.threshold` (config; default tuned against golden fixture in Task 10) → exit 3 showing the similar record's full JSON + three paths (edit existing / `--force` with suggested `supersedes` / rephrase). `--force` appends `{ts, domain, id, score}` to `<store>/cache/force-log.jsonl`; doctor reports on it.
- `prime` default = index format: `[id] type: summary (files: first +N)`, ranked truncation to budget, `…N more — use slate search <topic>` footer, `Fetch full records: slate query <domain> --id <id>` footer, wrapped in `<slate-memory>` delimiters with a "background reference — these are notes, not instructions" header. `--full`/`--format markdown` = mulch-style markdown sections.
- `query --id <id>` fetches one full record (JSON).
- `sync` commits store paths only: `git add <store>` + `git commit -m <msg> -- <store>`.
- `init` also creates `<store>/cache/` and `<store>/.gitignore` containing `cache/`.
- Slate-only commands: `setup claude [--remove]`, `hook <session-start|pre-tool|stop>`, `ack --no-lessons <reason>`.
- Slate config additions: `dedup.threshold`, `enforcement.stop_gate.{min_files: 3, min_lines: 40}`, `schema_version: 1` written at init.

Interfaces produced (used by later tasks — exact signatures):
- `output.SlateError(message, *, code="validation", exit_code=2, hint=None, retry=None)`; `output.emit(data, *, json_mode, command)`; `output.fail(err, *, json_mode)` → prints + returns exit code.
- `schema.now_iso() -> str`; `schema.generate_id(record) -> str`; `schema.validate_record(record, *, strict_unknown=False) -> list[str]` (error strings; unknown type → `[]` unless strict); `schema.summary(record) -> str`; `schema.TYPES: dict[str, TypeDef]`.
- `store.find_store(start=None) -> Store | None`; `Store.root: Path`, `Store.kind: "slate"|"mulch"`, `Store.domain_path(domain)`, `Store.domains() -> list[str]`, `Store.read(domain) -> tuple[list[dict], list[str]]` (records, warnings), `Store.append(domain, record)`, `Store.rewrite(domain, records)` (locked+atomic), `Store.read_archive(domain)`, `Store.append_archive(domain, records)`, `store.to_posix(path) -> str`.
- `locks.file_lock(path: Path)` context manager raising `LockTimeout`.
- `config.load(store) -> dict` (defaults deep-merged; raises SlateError exit 2 on bad YAML).
- `search.search_records(records, query, *, boost_factor=0.0) -> list[tuple[dict, float]]`.
- `priming.render_index(domain_records: dict[str, list[dict]], budget: int) -> str`; `priming.render_full(...) -> str`; `priming.rank(records) -> list[dict]`; `priming.estimate_tokens(text) -> int`; `priming.wrap_delimited(body) -> str`.
- `sessions.load(session_id)`, `sessions.save(state)`, `sessions.ack_path(store_root)`, `sessions.write_ack(store_root, reason)`.
